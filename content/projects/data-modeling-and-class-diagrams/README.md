## Your challenge

In this project you will practice **data modeling** and **class diagrams** to represent real systems clearly before implementing them.

You will work with [diagram.4geeks.com](https://diagram.4geeks.com/) instead of other external platforms. The goal is to identify entities, properties, data types, and relationships between models.

You must complete two exercises and deliver the results exported as PNG files.

---

## How to start the project

1. Open [diagram.4geeks.com](https://diagram.4geeks.com/).
2. Create a new diagram for each exercise.
3. Model the entities, their typed properties, and the relationships between them.
4. Export each result as PNG.

---

## Exercise 1: Music playlist player

Model a system similar to Spotify, where songs are individual objects living inside a playlist.

Requirements:

- It must have at least 5 models
- Every property must include its data type
- The related models must be clearly indicated
- The final result must be exported as PNG

Suggestion: think about models such as `User`, `Playlist`, `Song`, `Artist`, `Album`, or any other coherent variation.

---

## Exercise 2: Digital wallet

Model a digital wallet similar to Wise, including a transaction history.

Requirements:

- It must have at least 3 entities
- Every property must include its data type
- The related models must be clearly indicated
- The final result must be exported as PNG

Suggestion: think about models such as `Wallet`, `Account`, `Transaction`, `Currency`, `User`, or equivalent variants.

---

## What you need to do

- [ ] Create one diagram for the playlist player
- [ ] Create one diagram for the digital wallet
- [ ] Use data types in every property
- [ ] Define the relationships between models clearly
- [ ] Export both diagrams as PNG files

---

## What will be evaluated

- [ ] Clarity of the model
- [ ] Correct use of entities and relationships
- [ ] Presence of data types in properties
- [ ] Compliance with the minimum number of requested models or entities
- [ ] Overall coherence of the diagram

---

## How to submit this project

You must submit **two PNG images**:

- `music-playlist-class-diagram.png`
- `digital-wallet-class-diagram.png`

The delivery configuration expects **2 PNG files**.
